SETUP

  • Open index.html and change the information to your prefrence(more information on this below).
  • To set the timer and change other settings,open control.js. Set the timer by changing the 'date' value.
    The value is to be given in the format: 'date month year hour:miniute:second timezone'.
    EXAMPLE
     $('#CircleDown').circleDown({
   date: '12 May 2012 12:48:00 +0530'
   });
  • To set the subscription form and contact form,change the values inside suscribe.php and contact.php (located in the includes folder) as specified inside the files.
  • Upload the file to a php server.

Files and Folders

  • index.html: The file contains all the markup of the theme. It uses a slider(plugin) to display the information,slides are devided into list items,the list items contain an image with the class="aimg" which is set as the background image and other information belonging to the slide.
  • controls.js: The file has controls to set the timer and the slider.
  • styles.css: Contains all the styling information of index.html.
  • includes/contact.php The php to mail data entered in the contact form to your email ID.
  • includes/suscribe.php The php to mail the suscriber's email adress from the suscribe form to your email ID.
  • plugin/jQuery.js A javascript library that has useful functions that decrease file size and increases cross browser compatibility.
  • plugin/circleDown.js The script of the countdown plugin
  • plugin/circleDown.css The styling of the countdown plugin
  • plugin/openCanvas-packed.jsThe script of the the slider.
